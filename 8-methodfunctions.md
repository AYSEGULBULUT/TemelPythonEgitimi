```python
# moduller, kutuphaneler
import time

def yas_hesapla(dogum_yili):
    yas = suanki_yil - dogum_yili
    return yas

# simdiki yili bul
suanki_yil = int(time.strftime("%Y"))

# dogum tarihini iste
dogum_tarihi = int(input('Dogum tarihinizi giriniz.'))

yas = yas_hesapla(dogum_tarihi)

print(yas)
```
