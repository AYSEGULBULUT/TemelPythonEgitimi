### C vs Python
```python
# python
print('Hello, World!')
```
```c
// c
#include <stdio.h>
int main()
{
  printf("Hello, World!");
  return 0;
}
```
* **Daha iyi bir ornek**
```python
# python
year = 2021
print('Merhaba İZÜ!')
print(f"{year} Yılındayız.")
```
```c
// c
#include <stdio.h>

int main() {
	int year = 2021;
	printf("Merhaba İZU!\n")
	printf("%d Yilindayiz.\n", year);
	return 0;
}
```
