## Veri Tipleri

| Ornek | Veri Tipi
|-------|----------
|`x = "Merhaba IZU"`|String
|`x = 20` | Integer
|`x = 20.5` | Float
|`x = True , x = False` | Boolean
|`x = ['elma', 'armut']`| List
|`x = ('elma', 'armut')`| Tuple
|`x = {'adi': 'Nikita', ]'soyadi': 'Turkmen'}`| Dict
|`x = {'elma', 'armut'}`| Set
